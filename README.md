# Titanic Data Analysis 🚢📊

This project performs **Exploratory Data Analysis (EDA)** on the famous Titanic dataset using Python.
The goal is to analyze passenger information and visualize survival patterns.

---

## 📌 Project Overview

The Titanic dataset contains information about passengers such as:

* Passenger ID
* Age
* Gender
* Passenger Class
* Fare
* Cabin
* Embarked Port
* Survival Status

Using this dataset, we perform **data cleaning, preprocessing, and visualization**.

---

## 🛠 Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib

---

## 📂 Dataset

Dataset used: **Titanic Train Dataset**

Columns included in the dataset:

* PassengerId
* Survived
* Pclass
* Name
* Sex
* Age
* SibSp
* Parch
* Ticket
* Fare
* Cabin
* Embarked

---

## 🔎 Steps Performed in the Project

1. Loaded the dataset using **Pandas**
2. Explored dataset structure using `info()` and `describe()`
3. Checked for **missing values**
4. Replaced null values using `fillna()`
5. Checked for **duplicate records**
6. Created **data visualizations**

---

## 📊 Visualizations

The following graphs were created:

### 1️⃣ Survival Count (Bar Graph)

Shows how many passengers survived and did not survive.

### 2️⃣ Age Distribution (Histogram)

Displays the distribution of passenger ages.

### 3️⃣ Gender Distribution (Pie Chart)

Shows the percentage of male and female passengers.

```

```

4. Run the Python script:

